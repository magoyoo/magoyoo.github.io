1.0.4 블로그 업데이트

 

 

1.0.4 업데이트 내용 📚
  • [CHANGED] 최신글 UI 변경
  • [FEATURE] Search 기능 추가
  • [FEATURE] Tags Detail 페이지 신설
  • [FEATURE] Floating Tag Items Components 추가

 

  본래 예정이었던 "모바일 대응" 작업은 뒤로하고 Search 기능과 Tag 상세화면 작업을 미리 수행하였다. 역시 기능 위주로 작업하는 것이 더 재미있다. 검색기능은 화면 요소가 많지 않아 시간이 오래 걸리지 않았다. 다만 추후 글이 많아졌을 때 성능이 걱정된다. 비록 아주 먼 미래의 일이지만. 그때가 되면 SSG 방식이 아닌 SSR 방식으로 재구현해봐야겠다.

  태그 모음 floating 영역은 IE(Internet Explore) 대응을 할 필요가 없어져서 sticky display로 쉽게 구현했다. 항상 웹 개발은 브라우저 대응, 특히 IE 때문에 골머리를 싸매야 했지만, 이제 더 이상 IE 고려를 안 해도 되어 기쁘다.

  UI는 기존에 만들었던 컴포넌트를 재사용하였다. 요즘 많이 사용하는 블로그를 보면 같은 기능을 하는 컴퍼넌트들이 각 화면의 위치와 배치에 따라 다르게 디자인되어있는 것을 확인할 수 있다. 단순히 사용성을 위해서라면 규칙적인 패턴으로 예상 가능한 동작들을 제공하는 것이 더 올바르다 생각되지만, 너무 똑같은 패턴은 지루한 경험을 줄 수 있다. 사용성을 극대화하면서 지루한 경험을 주지 않게 만드는 것이 UX의 설계일까? 아직 그 영역은 잘 모르겠다. 다만 추후 업데이트에서는 태그 상세 화면, 검색 상세 화면, 메인 카테고리 상세 화면에서 쓰이고 있는 컴포넌트의 디자인을 조금씩 변경해볼 생각이다.

1.0.5 예정
  • 모바일 대응
  • About 페이지 신설
  • 블로그 컴포넌트 디자인

 

  이제 디테일한 작업만 남았다. 특히 About 페이지는 만들겠다 마음만먹었지, 세부적인 계획은 아직 없다. 마무리 작업이 너무 늘어지면 블로그 리뉴얼 프로젝트는 올해 안 끝날 것이다. 이 작업을 잘 마무리해야 앞으로 글도 더 많이 쓸 수 있을 것이기에 빠르게 마무리해야겠다.
